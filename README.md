# LaAscencion: Landing Page B2B para Serviços Funerários

## 1. Visão Geral do Projeto

Este projeto consiste na criação de uma *Landing Page* focada no modelo de negócios B2B (Business-to-Business) para a prestação de serviços funerários. O desenvolvimento foi realizado como um exercício prático  e avaliativo, com o objetivo de demonstrar a competência na estruturação e estilização de páginas web estáticas.

O foco principal é apresentar a empresa parceira de forma profissional e acessível, direcionando a comunicação para outras entidades corporativas (como hospitais, casas de repouso e seguradoras) que necessitam de uma solução terceirizada e confiável.

### Objetivo Central

* Demonstrar proficiência na utilização das tecnologias fundamentais da web (HTML e CSS).
* Estruturar uma interface de usuário clara e navegável, mantendo um tom de comunicação formal, condizente com a natureza sensível do nicho de mercado.

## 2. Tecnologias Utilizadas

* **HTML5:** Empregado para a estruturação semântica de todo o conteúdo da página, garantindo acessibilidade e organização.
* **CSS3:** Utilizado para a estilização, controle de layout (ex: Flexbox ou Grid, se aplicável) e para garantir a responsividade (ajuste da página a diferentes dispositivos).

### Observação sobre Conteúdo

Apenas algumas **imagens conceituais** foram geradas com o auxílio de ferramentas de Inteligência Artificial, servindo como base visual para o *mockup* do projeto. Todo o código (HTML e CSS) foi escrito manualmente por mim.

## 4. Aprendizados e Desafios

### Principais Aprendizados:

* **Estrutura Semântica:** Reforço na importância da correta utilização das tags HTML5 para a organização lógica e para a otimização de mecanismos de busca.
* **Design Consistente:** Domínio sobre o uso de variáveis CSS e *reset* de estilos para manter a uniformidade visual em toda a página.
* **Tom de Voz B2B:** Habilidade de traduzir um tema sensível (serviços funerários) em um design e conteúdo que transmitem **confiança, seriedade e profissionalismo** para um público corporativo.

### Desafios Superados:

* **Responsividade:** A garantia de que o layout e a tipografia fossem adequados para visualização tanto em *desktops* quanto em dispositivos móveis, sem comprometer a legibilidade e o peso visual.
* **Gerenciamento de Assets:** Organização eficiente dos arquivos CSS e das imagens (incluindo as geradas por IA) dentro da estrutura de pastas do projeto.
